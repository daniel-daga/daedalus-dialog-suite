# Daedalus Dialog Suite

Monorepo for the Daedalus dialog tooling stack:

- `daedalus-dialog-editor` - Electron-based visual editor for dialogs and quests
- `daedalus-parser` - Tree-sitter based parser and semantic/codegen library
